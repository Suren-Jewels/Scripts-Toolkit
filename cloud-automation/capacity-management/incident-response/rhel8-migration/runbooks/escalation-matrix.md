# Escalation Matrix (Migration Incidents)

## Severity Levels
- **SEV‑1 (P1)** — Full outage, customer impact  
- **SEV‑2 (P2)** — Major degradation  
- **SEV‑3 (P3)** — Minor impact  
- **SEV‑4 (P4)** — Informational  

## Contacts by Severity

### SEV‑1
- On‑call SRE  
- Incident Commander  
- Network Lead  
- Platform Engineering Manager  

### SEV‑2
- SRE Team  
- Application Owner  
- Migration Program Lead  

### SEV‑3
- Application Support  
- Migration Engineer  

### SEV‑4
- Documentation / Backlog
